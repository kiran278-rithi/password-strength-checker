# Password Strength Checker

## Problem Statement
Many users create weak passwords that are easy to guess, putting their accounts at risk.  
This project helps users check the strength of their passwords and encourages stronger, safer passwords.

## Features
- Checks password length  
- Detects numbers in the password  
- Detects special characters (like @, #, !)  
- Gives feedback: Weak ❌, Medium ⚠️, Strong ✅
